Editing preferences

 

 

If you are going through a proxy server for Web content, then you need to specify this when you launch Agilent Literature Search for the first time. 

 

 

You will need to specify the host name, the port upon which the proxy server is connected to, and whether or not the proxy is enabled.   Setting the enabled attribute allows you to more easily switch between different computing environments without having to re-enter host information.   The figure below shows an example of filled in proxy settings (Note: this is a fictitious host name and port):

 

You can also modify  proxy settings via the Edit -> Preferences ... menu item.

 

Note: if you don't know whether you are using a proxy server, or don't know the name and/or port of your proxy server, you can usually find out by looking at the Preferences of your Web browser.    For example, if you are using Microsoft Internet Explorer, select the Tools -> Internet Options menu item, then select Connections tab, then press LAN Settings... button.  If you are using a proxy server, you will see the proxy server Address and Port filled in and a checkmark in the checkbox entitled "use a proxy server".
