Saving and restoring association networks

 

Association networks generated by Agilent Literature Search are bona fide Cytoscape networks and can be saved and re-loaded in the same way as any other Cytoscape network.

However, it is not possible under the current version of Cytoscape to save and restore the sentences and literature references that Agilent Literature Search attaches as Cytoscape Node and Edge attributes.  To enable you to do this, the Agilent Literature Search plugin provides operations for saving and restoring association networks. 

You can save an Agilent Literature Search network by selecting the File -> Export -> as Agilent Literature Search Network menu item in Cytoscape.

 

You will be prompted to choose a file name and location to save to.

 

 

 

Agilent Literature Search uses the file extension .alfa to distinguish files containing Alfa Literature Search networks.  If you don't specify a .alfa extension, then it will be added to your specified file name by default.

 

To restore a previously saved Agilent Literature Search network, select the File -> Import -> Agilent Literature Search Network menu item in Cytoscape

You will be prompted to choose a file with .alfa extension to load into Cytoscape.

 

New Cytoscape Session Save (v2.3)
Cytoscape 2.3 enables users to save a Cytoscape session. The Agilent Literature Search networks and associated node and edge attributes are compatible with the Cytoscape session saving and loading features. If a Cytoscape session that contains networks obtained by the Agilent Literature Search tool is saved, then all the Agilent Literature Search tool networks will also be loaded properly. For further information regarding Cytoscape Session Save, please refer to the Cytoscape User Manual.
