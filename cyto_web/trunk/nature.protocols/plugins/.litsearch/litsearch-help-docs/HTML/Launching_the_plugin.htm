Launching the plugin

You can launch the plugin by selecting the Agilent Literature Search item on Cytoscape's Plugins menu. 

When you launch Agilent Literature Search, you will be prompted to accept the Third Party Software Copyright Notice before you can proceed.

You can supress this dialog from coming up in the future by checking the checkbox labeled "Don't show this dialog again".

The first time you run Agilent Literature Search, you will be asked if you use a proxy server.  If you answer yes, then you will be prompted for a host name and port number.  This is described in the section on Editing Preferences.
