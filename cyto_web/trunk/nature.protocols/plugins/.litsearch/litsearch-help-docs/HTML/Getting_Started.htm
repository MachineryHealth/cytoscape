Getting Started

This section has instruction for installing Agilent Literature Search plugin for Cytoscape v2.3 and for launching the Agilent Literature Search plugin from Cytoscape v2.3.

 

 

System requirements

Currently, Agilent Literature Search runs under Java on Linux, Windows, and Mac OS X.  The system requirements for Agilent Literature depend on the size of the networks the user wants to generate and then load, view and manipulate in Cytoscape. We recommend a recent computer (1.5GHz CPU or higher) with at least 512MB of free physical RAM.    Like Cytoscape, Agilent Literature Search expects a minimum screen resolution of 1024x768.
