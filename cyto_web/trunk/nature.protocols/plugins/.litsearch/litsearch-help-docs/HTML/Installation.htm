Installation

You can install Agilent Literature Search from the Web page at http://www.cytoscape.org/download_agilent_literature_search_v2.3.php?file=litsearch_v2.3.

Follow the instructions given by the InstallAnywhere process that is invoked when you select the link to download Agilent Literature Search. 

You will be prompted to accept the Agilent Technologies Software License Agreement in order to continue installing Agilent Literature Search. 

You will also be asked to choose a directory for installing Agilent Literature Search.  You should choose the plugins directory in wherever you have installed Cytoscape.

The installer will copy a num ber of files into a directory called .litsearch, which is created under your <user.home> directory.  You will not need to alter any of these files unless you want to customize the behavior of Agilent Literature Search.  This is discussed in the sections on Configuring Literature Search, and Working with Lexicons .

 

Uninstalling Agilent Literature Search

When you install Agilent Literature Search, an uninstaller program is also installed into the directory <user.home>/.litsearch/.    There will be a file in that directory named

Uninstall Agilent_Literature_Search.exe

If you execute that file, then the Agilent Literature Search plugin and all of its associated files will be removed from the system.  Note, however, that the uninstaller will not delete any files that were not created by the installer.  So, for example, it will not delete any literature search network files that you may have saved.  If you want to delete such files, you will need to do that manually.
