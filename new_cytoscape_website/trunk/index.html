Where to Start?

Biology

Cytoscape was developed for biological researches, and it supports a lot of usecases including:

  • Load protein-protein interaction data sets in various formats
  • Integrate annotation data, such as Gene Ontology, to the network
  • Analyze protein-protein/genetic interaction network with plugins
  • Map experimental data (e.g. microarray) to your network data sets and visualize it
  • Visualize and analyze human-curated pathway datasets such as KEGG.

Social Science

You can analyze and visualize network of people interactively on Cytoscape:

  • Import your network data from spreadsheet
  • Write your own web crawler in scripting languages within Cytoscape
  • Calculate network statistics using plugins

Semantic Web

Since semantic web data sets are esentially graphs, you can use Cytoscape as a workbench for RDF data:
  • Load and visualize RDF
  • Reasoning by plugins
  • Use as a platform to develop your own semantic web applicaitons

General Complex Network Analysis

Cytoscape is a domain-independent platform, and you can use it as a powerful tool for complex network analysis in general.

  • Calculate statistics for networks
  • Import data files from other network analysis tools such as igraph, pajek, or graphviz (by standard file formats like GML)

Plugin Development

Cytoscape is an expandable platform. You can write your own plugins to implement new features for Cytoscape.

  • Build Cytoscape from source code
  • Develop a plugin
  • Cytoscape API
