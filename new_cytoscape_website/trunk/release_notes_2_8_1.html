Updates in Cytoscape 2.8.1

Bug Fixes

  • "Select All Attributes" button on the data panel selects the attributes in a sorted order.
  • Added the ability to load a plugin from a local file
  • Changed internal structures for group viewers to allow group viewer information about a group to be tracked before the group viewer is actually registered
  • Added a "discrete layout" setting for ForceDirected so that the force directed layout will now always give the same answer
  • Cleaned up XGMML group reading and writing
  • Several fixes to group handling
  • Minor fixes to tunable presentations
