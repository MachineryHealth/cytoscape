NetLink Plugin

The NetLink plugin is a simple plugin for connecting nodes or edges in one network with another network. The basic idea is to create an attribute which point to the names of the other network. The NetLink plugin simply looks for an attribute named netlink.connection or and if that attribute exists for the given node or edge, it then creates a menu item in the right-click (context) popup menu. If the menu item exists, then chosing the menu item will switch the selected view focuse from the current network to the new network.

Parent -> Child Links

Begin with a simple network called Parent.sif: 

A pp B
B pp C
C pp A

Next, assume that node A points to child network A.sif, B points to child network B.sif, and C points to child network C.sif.

A.sif: 

one pp two
two pp three

B.sif: 

marge knows homer
lisa knows bart

C.sif: 

cat xx dog
bird xx fish

Now, to create the links from the Parent nodes to the child nodes, create a node attribute file where the attribute is named netlink.connnection. 

netlink.connection
A = A.sif
A = B.sif
A = C.sif

Now load all networks and the node attributes file. When you right click on a node in the Parent.sif network, you should see a menu item that says: "asdf".

Child -> Parent Links

If you want to add Child ->Parent links, then simply append the node names found in the child networks the node attributes file that you just created: 

netlink.connection
A = A.sif
B = B.sif
C = C.sif
one = Parent.sif
two = Parent.sif
three = Parent.sif
cat = Parent.sif
dog = Parent.sif
...
