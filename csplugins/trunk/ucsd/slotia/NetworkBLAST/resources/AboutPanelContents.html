NetworkBLAST

What does NetworkBLAST Do?

The NetworkBLAST Plug-in provides facilities for comparative analysis of two protein networks within Cytoscape. The plug-in uses the Network Comparison Toolkit (NCT) Library for performing the comparative algorithms. The NCT library employs algorithms based on PathBLAST research (1).

NetworkBLAST aligns two protein interaction graphs represented by Cytoscape networks to highlight strongly homologous pathways. The results are Cytoscape networks whose nodes represent homologous proteins and whose edges represent the probability of the homology.

Using NetworkBLAST

Before Using NetworkBLAST: Having the Right Input Graphs

Input Graphs

The edges of each input graph must have the interaction attribute that defines the probability of the protein interaction. It must have the probability stored as a string.

SIF Files: Each line in a SIF file defines an edge. Each line has the format:
name_of_source_node interaction_type name_of_target_node
Storing the probability of the interaction is done by replacing interaction_type with the numerical value of the probability. For example: 

node1 0.07261985 node2
The above line defines the edge between node1 and node2 with the interaction probability of 0.07261985. When the SIF file is imported into Cytoscape, it will properly parse the probability and store it in the interaction attribute.
