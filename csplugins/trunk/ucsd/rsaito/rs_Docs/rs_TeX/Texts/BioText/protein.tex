\section{タンパク質の化学}

\begin{figure}
\begin{center}
\includegraphics{Graphix/aminoacid1.eps}
\end{center}
\caption{タンパク質を構成する20種のアミノ酸の構造式と表記法}
\label{aminoacid1}
\end{figure}

タンパク質\index{たんぱくしつ@タンパク質}はアミノ酸\index{あみのさん@ア
ミノ酸}が鎖状につながった構造をしていますが、構成するアミノ酸の種類や並
び方、長さによって多様な機能を持つことができます。図\ref{aminoacid1}にタ
ンパク質を構成する20種類\footnote{近年ではセレノシステインやピロリシンな
ど細胞内で合成される新たなアミノ酸も発見されている\cite{pyrro}。}のアミノ酸の構造式
と名称、表記法を示します。タンパク質の中のアミノ酸は、そのカルボキシル基
(-COOH)と隣のアミノ酸のアミノ基(NH2-)が脱水縮合したペプチド結合によって
鎖状につながっています。アミノ酸はその側鎖の種類によりおおまかに以下の4
種類に分類できます。

\begin{enumerate}
\item 疎水性アミノ酸（Ara, Val, Leu, Ile, Met, Trp, Phe, Pro）
\item 極性無電荷アミノ酸（Gly, Ser, Thr, Cys, Tyr, Asp, Gln）
\item 正電荷をもつ極性アミノ酸（Lys, His, Arg）
\item 負電荷をもつ極性アミノ酸（Asp, Glu）
\end{enumerate}

\noindent
各分類に属するアミノ酸同士は、側鎖の大きさに違いがありますが、化
学的な性質は似ているということができます。

タンパク質の構造は4つの段階によって決定されています。1次構造とはアミノ
酸の配列のことを指します。2次構造とは、連続するアミノ酸が局所的にとる立
体構造を指し、\(\alpha\)-へリックス（らせん）や\(\beta\)-シート構造などが知られています。
3次構造は、それらの2次構造が組み合わさって一本のペプチド鎖としてどのよ
うな全体構造をしているかを示します。そして4次構造とは、3次構造をとった
ペプチド鎖がさらに高次に組み合わさって複合体をつくるときの、その組み合わ
せ構造をさします。このようにアミノ酸が連結してできたタンパク質は、非常に
複雑な構造を持っていて(図\ref{prot_str1})、様々な細胞内反応の触媒機能や他の物質に結合する機能などを
司っています。タンパク質の構造と機能の研究は、現在多くの研究
者が参入している分野の一つで、最近の10年ほどで立体構造の解かれたタン
パク質の数は指数関数的に多くなっています\footnote{日本でも2002年から5年間理化学研究所でタンパク3000という
大規模なタンパク質構造決定プロジェクトが進められた。}。
