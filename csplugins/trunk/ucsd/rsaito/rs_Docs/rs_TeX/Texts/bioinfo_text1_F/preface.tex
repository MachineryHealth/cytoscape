\chapter*{まえがき}

このドキュメントは，サイエンス社の数理科学別冊用クラスファイル
（{\ttfamily msse.cls}）の使い方を説明したものです．

以下のような構成になっています．

\begin{itemize}
\item
まえがき（{\ttfamily preface.tex}）

\item
第1章（{\ttfamily readme-1.tex}）\par
本クラスファイルの使い方や特徴，テキストを記述する際の
注意点などをまとめています．

\item
第2章（{\ttfamily readme-2.tex}）\par
美しい組版を行うためにタイピングに際しての注意点や，
数式が版面からはみ出す場合の対処方法などをまとめてあります．

\item
付録A（{\ttfamily readme-3.tex}）\par
本クラスファイルに組み込んでいるマクロおよび
{\ttfamily msse-mac} パッケージで利用できるコマンドや環境を
説明しています．

\item
参考文献（{\ttfamily refer.tex}）．
\TeX{} 関連の著書を紹介しています．
文献\Cite{texbook}から\Cite{Salomon}までが \TeX{} や \LaTeX 2.09 の
解説書で，\Cite{Nakano} 以下が \LaTeXe{} の解説書です

\item
索引（{\ttfamily readme.ind}）．
サンプルとして，「徹底研究 ソフトウェア」（サイエンス社刊）の
データを借用しました．
\end{itemize}

このドキュメントは\LaTeX{} や\LaTeXe{} の解説書ではありません．
すでに何らかの解説書を読まれていることを前提にしています．

クラスファイルおよびドキュメントの作成は
FreeBSD 4.2 上の mule 2.3 + yatex 1.67 の環境で行いました．
