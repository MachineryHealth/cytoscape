\section{積分}

\subsection{不定積分の定義}

\(F(x)\)の導関数が\(f(x)\)であるとき、すなわち\(F'(x)=f(x)\)であるとき、
\(F(x)\)を\(f(x)\)の{\bf 原始関数}といい、\(f(x)\)の原始関数全体を\(f(x)\)の{\bf 不定積分}という。
後者を\(\mathscr{F}(x)\)で表すとき(\(F(x) \in \mathscr{F}(x)\))、

\begin{displaymath}
\int f(x)dx=\mathscr{F}(x)
\end{displaymath}

\noindent
と表現する。あるいは\(C\)を任意の定数として、

\begin{displaymath}
\int f(x)dx=F(x)+C
\end{displaymath}

\noindent
と表現してもよい。
\(f(x)\)の不定積分を求めることを\(f(x)\)を{\bf 積分}するという。

\subsection{変数のべき乗の積分}

$n\neq 1$の実数のとき、

\begin{displaymath}
\int x^{n}dx=\frac{1}{n+1}x^{n+1}+C
\end{displaymath}

\subsection{置換積分法}

$x=g(t)$のとき、

\begin{displaymath}
\int f(x)dx=\int f(g(t))g^{\prime}(t)dt
\end{displaymath}

$g(x)=t$のとき、

\begin{displaymath}
\int f(g(x))g^{\prime}(x)dx=\int f(t)dt
\end{displaymath}

\subsection{部分積分法}

\begin{displaymath}
\int f^{\prime}(x)g(x)dx=f(x)g(x)-\int f(x)g^{\prime}(x)dx
\end{displaymath}


\subsection{定積分の定義}

\begin{figure}
\begin{center}
\includegraphics[scale=0.5]{Figures/integral1}
\end{center}
\vspace{-2em}
\caption{定積分の幾何学的意味}
\label{integ1}
\end{figure}

図\ref{integ1}に示すように、
区間\([a,b]\)で連続な関数\(f(x)\)に関して、
\(x=a\)から\(x=b\)の位置までを\(n\)等分すると\footnote{話を分かりやすくするために等分にしているが、実際は等分にする必然性はない。}、
各区間の大きさ\(\Delta x\)は\(\frac{b-a}{n}\)になる。今分割位置
\(x_{1}=a,x_{2}=a+\Delta x,x_{3}=a+2\Delta x, \cdots, x_{n}=a+(n-1)\Delta x=b-\Delta x\)および、どの位置における
\(f(x)\)の値すなわち、\(f(x_{1}),f(x_{2}),\cdots,f(x_{n})\)に注目し、\(\displaystyle \sum_{i=1}^{n}f(x_{i})\Delta x\)を考えると、
これは図\ref{integ1}の灰色の長方形の面積の合計に等しい。ここで\(n\)が限りなく大きくなるとき、すなわち\(\Delta x\)が限りなく
小さくなるときに、この式の値が有限確定値に近づくなら、この極限を

\begin{displaymath}
\int_{a}^{b}f(x)dx=\lim_{n\to\infty}\sum_{i=1}^{n}f(x_{i})\Delta x=\lim_{\Delta x\to 0}\sum_{i=1}^{n}f(x_{i})\Delta x
\end{displaymath}

\noindent
と表し、\(\int_{a}^{b}f(x)dx\)を\(f(x)\)の\(a\)から\(b\)までの{\bf 定積分}(あるいは略して積分)という。
図\ref{integ1}から想像できるように、\(\Delta x\)が小さくなると、長方形の面積の総計は\(f(x)\)の\(a\)から\(b\)までの曲線
から下の領域の面積に近づく。すなわち、\(\int_{a}^{b}f(x)dx\)は\(f(x)\)の\(a\)から\(b\)までの曲線から下の領域の面積を表している。

\(f(x)\)の原始関数を\(F(x)\)とすると、

\begin{displaymath}
\int_{a}^{b}f(x)dx=F(b)-F(a)
\end{displaymath}

\noindent
の関係が成り立つ。

\subsection{定積分の置換積分法}

$x=g(t), a=g(\alpha), b=g(\beta)$のとき、

\begin{displaymath}
\int_{a}^{b}f(x)dx=\int_{\alpha}^{\beta}f(g(t))g^{\prime}(t)dt
\end{displaymath}

\subsection{定積分の部分積分法}

\begin{displaymath}
\int_{a}^{b}f^{\prime}(x)g(x)dx=\left[f(x)g(x)\right]_{a}^{b}-\int_{a}^{b}f(x)g^{\prime}(x)dx
\end{displaymath}

\subsection{定積分と不定積分の関係}

$f(x)$を連続関数とすれば、$a$が定数のとき、

\begin{displaymath}
\frac{d}{dx}\int_{a}^{x}f(t)dt=f(x)
\end{displaymath}
