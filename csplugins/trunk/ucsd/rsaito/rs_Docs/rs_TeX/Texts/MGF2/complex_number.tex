\section{複素変数の関数とその正則性}

\label{complex_number}

\subsection{複素変数の関数}

\(i\)を{\bf 虚数単位}(\(i^{2}=-1\))とし、\(a,b\)を実数としたとき、\(\alpha=a+bi\)を{\bf 複素数}(complex number)ということを思い出そう。
\(a\)と\(b\)をそれぞれ複素数\(\alpha\)の実部と虚部といい、\(\mbox{Re}(\alpha)=a,\mbox{Im}(\alpha)=b\)で
表す。また\(\bar{\alpha}=a-bi\)を\(\alpha\)の{\bf 共役複素数}または共役数という。

\begin{figure}
\begin{center}
\includegraphics[scale=0.8]{Figures/complex_xy1}
\end{center}
% \vspace{-2em}
\caption{複素数と複素数平面}
\label{complex_xy1}
\end{figure}

実数が直線上のある1点を表すものとすれば、複素数は図\ref{complex_xy1}に示すように、\(xy\)座標を持つ平面上の1点P\((x,y)\)を表すと考えることができる。
この1点に対応する複素数\(z\)は\(z=x+yi\)である。図の例では\(x=a, y=b\)になっている。
このように点が複素数を表しているような平面を{\bf 複素平面}\index{ふくそへいめん@複素平面}またはガウス平面という。
複素数の絶対値はベクトルOPの長さと定義され、\(|a+bi|=\sqrt{a^{2}+b^{2}}\)で表される。

複素変数\(w\)が\(w=u+vi\)(但し\(u,v\)は実数)と表され、複素変数\(u,v\)がさらに\(u=u(x,y)\)、\(v=v(x,y)\)(但し\(x,y\)は実数)と
表されるとき、\(w\)は\(z\)の関数であるといい\footnote{\(z\)は\((x,y)\)の組み合わせで\(z=x+yi\)と一意に決まることに注意。}、

\begin{equation}
w=f(z)
\end{equation}

\noindent
などの記号を用いて表す。


\subsection{正則関数}

複素変数\(z\)の関数\(f(z)\)の定義域内に点\(z=a\)を考えて独立変数\(z\)が点\(a\)に近づくとき、その近づき方に
関係なく極限
\(\lim_{z\to a}\frac{f(z)-f(a)}{z-a}\)
が確定して存在しているとき、\(w=f(z)\)は点\(z=a\)で微分可能であるといい、これを

\begin{equation}
f'(a)=\frac{df(a)}{dz}=\lim_{z\to a}\frac{f(z)-f(a)}{z-a}
\label{eq_complex123}
\end{equation}

\noindent
と表す。また\(\Delta z=z-a\)、\(\Delta w=(f(z)-f(a)\)とおくと、
式\ref{eq_complex123}を

\begin{equation}
\lim_{\Delta z\to 0}\frac{\Delta w}{\Delta z}=\lim_{\Delta z\to 0}\frac{f(a+\Delta z)-f(a)}{\Delta z}
\label{eq_complex124}
\end{equation}

\noindent
と書くこともできる。ここで\(w=f(z)\)が点\(z=a\)で微分可能であるとは、\(\Delta z\)が0に近づけば
その経路に関わらず式\ref{eq_complex124}が確定した値になることである。
今\(z\)が取る値の範囲を領域\(D\)で表すと、\(D\)内の全ての点で関数\(w=f(z)\)が微分可能であるとき、
関数\(w=f(z)\)は領域\(D\)で{\bf 正則}\index{せいそく@正則}であるといい、
\(w=f(z)\)を{\bf 正則関数}\index{せいそくかんすう@正則関数}という。このとき、領域\(D\)の各点\(z\)での微分係数
\(f'(z)\)を考えることによって、\(w=f(z)\)の導関数が得られる。導関数については、実変数の関数の場合と
同じ記号を用いる。

\vspace{1em}

\refstepcounter{preexer}

\noindent
{\bf 例題\thechapter-\thepreexer}:~ 
\(w=z^{2}=(x+yi)^{2}\)のとき、\(w\)を微分しよう。

\begin{displaymath}
f'(z)=\lim_{\Delta z\to 0}\frac{\Delta w}{\Delta z}
=\lim_{\Delta z\to 0}\frac{(z+\Delta z)^{2}-z^{2}}{\Delta z}=\lim_{\Delta z\to 0}(2z+\Delta z)=2z
\end{displaymath}

\vspace{1em}

\refstepcounter{exer}
\noindent
{\bf 問題\thechapter-\theexer}:~ 次の関数\(w\)を微分せよ。
\begin{displaymath}
w=\frac{z}{z-i}=\frac{(x+yi)}{x+yi-i}
\end{displaymath}

\refstepcounter{exer}
\noindent
{\bf 問題\thechapter-\theexer}:~ 次の関数\(w\)は微分できないことを示せ。

\begin{displaymath}
w=2x+yi
\end{displaymath}

\subsection{コーシー・リーマンの方程式}

複素変数\(z=x+yi\)の関数\(f(z)=u(x,y)+v(x,y)i\)が微分可能であるための条件を求めよう。
実変数\(t\)の関数
\(\varphi(t)=\frac{at+b}{ct+d}\)
が定数であるための必要十分条件は、\(ad-bc=0\)である。なぜならば
\(\varphi'(t)=\frac{ad-bc}{(ct+d)^{2}}\)
であるからである。これを用いて以下の定理を導こう。

\vspace{1em}

\refstepcounter{theorem}

\noindent
{\bf 定理\thechapter-\thetheorem}:~ 領域\(D\)で定義された関数\(f(z)=u(x,y)+v(x,y)i\)
について、\(u(x,y)\)と\(v(x,y)\)が連続な偏導関数を持つとする。このとき
関数\(f(z)\)が正則であるための必要十分条件は、

\begin{equation}
\frac{\partial u}{\partial x}=\frac{\partial v}{\partial y}
\ \ \ \mbox{かつ} \ \ \ 
\frac{\partial u}{\partial y}=-\frac{\partial v}{\partial x}
\label{eq_cauchy_riemann10}
\end{equation}

\noindent
{\bf 証明\thechapter-\thetheorem}:~
$w=f(z)$とする。$\Delta z=\Delta x+\Delta yi, \Delta w=f(z+\Delta z)-f(z)$とすれば、

\begin{displaymath}
\frac{\Delta w}{\Delta z}=\frac{f(z+\Delta z)-f(z)}{\Delta z}=\frac{\{u(x+\Delta x,y+\Delta y)+v(x+\Delta x,y+\Delta y)i\}-\{u(x,y)+v(x,y)i\}}{\Delta x+\Delta yi}
\end{displaymath}

\noindent
ゆえに、

\begin{equation}
\frac{\Delta w}{\Delta z}=\frac{\{u(x+\Delta x,y+\Delta y)-u(x,y)\}+\{v(x+\Delta x,y+\Delta y)-v(x,y)\}i}{\Delta x+\Delta yi}
\label{eq_cauchy_riemann_d1}
\end{equation}

\noindent
さて$u(x,y)$と$v(x,y)$が連続な偏導関数を持つから、これらは全微分可能である。ゆえに

\begin{eqnarray}
u(x+\Delta x,y+\Delta y)-u(x,y) &=& u_{x}\Delta x+u_{y}\Delta y+\epsilon_{1}\label{eq_cauchy_riemann_d2_1}\\
v(x+\Delta x,y+\Delta y)-v(x,y) &=& v_{x}\Delta x+v_{y}\Delta y+\epsilon_{2}\label{eq_cauchy_riemann_d2_2}
\end{eqnarray}

\noindent
となる。$\rho$=$\sqrt{(\Delta x^{2})+(\Delta y)^{2}}$として

\begin{equation}
\frac{\epsilon_{1}}{\rho}\rightarrow 0,\ \ \ \ \frac{\epsilon_{2}}{\rho}\rightarrow 0\ \ \ \ \ (\Delta x\rightarrow 0, \Delta y\rightarrow 0) \label{eq_cauchy_riemann_d3}
\end{equation}

\noindent
式\ref{eq_cauchy_riemann_d2_1},\ref{eq_cauchy_riemann_d2_2}を式\ref{eq_cauchy_riemann_d1}に代入して
$m_{*}=\frac{\Delta y}{\Delta x}$とおけば、

\begin{displaymath}
\frac{\Delta w}{\Delta z}=\frac{(u_{x}\Delta x+u_{y}\Delta y+\epsilon_{1})+(v_{x}\Delta x+v_{y}\Delta y+\epsilon_{2})i}{\Delta x+\Delta yi}
\end{displaymath}

\noindent
従って

\begin{equation}
\frac{\Delta w}{\Delta z}=\frac{(u_{x}+u_{y}m_{*})+(v_{x}+v_{y}m_{*})i}{1+m_{*}i}+\epsilon
\label{eq_cauchy_riemann_d4}
\end{equation}

\noindent
ここで

\begin{displaymath}
\epsilon=\left(\frac{\epsilon_{1}}{\rho}\frac{\Delta x}{\rho}+\frac{\epsilon_{2}}{\rho}\frac{\Delta y}{\rho}\right)+\left(\frac{\epsilon_{2}}{\rho}\frac{\Delta x}{\rho}-\frac{\epsilon_{1}}{\rho}\frac{\Delta y}{\rho}\right)i
\end{displaymath}

\noindent
である。
さて$\left|\frac{\Delta x}{\rho}\right|\leq 1, \left|\frac{\Delta y}{\rho}\right|\leq 1$であるから、
式\ref{eq_cauchy_riemann_d3}によって

\begin{equation}
\epsilon\rightarrow 0\ \ \ \ \ (\Delta x\rightarrow 0, \Delta y\rightarrow 0)
\label{eq_cauchy_riemann_d5}
\end{equation}

\noindent
である。
今点$z$を通る曲線$C$を任意にとる。曲線$C$に沿って点$z+\Delta z$が点$z$に近づくとすれば、曲線$C$上の点$z$における
接線の傾きを$m$として、

\begin{equation}
m_{*}=\displaystyle \frac{\Delta y}{\Delta x}\rightarrow m
\end{equation}

\noindent
である。ゆえに式\ref{eq_cauchy_riemann_d4}と\ref{eq_cauchy_riemann_d5}によって

\begin{equation}
\frac{\Delta w}{\Delta z}\rightarrow\frac{(u_{y}+v_{y}i)m+(u_{x}+v_{x}i)}{im+1}
\end{equation}

\noindent
さて$w=f(z)$が微分可能であるための必要十分条件は

\begin{displaymath}
\frac{(u_{y}+v_{y}i)m+(u_{x}+v_{x}i)}{im+1}
\end{displaymath}

\noindent
がどんな$m$に対しても同一の値であることである。従ってこの必要十分条件は

\begin{displaymath}
(u_{y}+v_{y}i)\cdot 1-(u_{x}+v_{x}i)\cdot i=0
\end{displaymath}

\noindent
ゆえに$(u_{y}+v_{x})+(v_{y}-u_{x})i=0$となり、
従って$u_{x}=v_{y},  u_{y}=-v_{x}$が成立する。
