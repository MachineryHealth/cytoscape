\section{ディラックのデルタ関数}

\label{delta_func}

既に節\ref{prob_distr}のサイコロやサイコロルーレットの話で触れているが、
一定の範囲で一定の確率密度(離散確率の場合は単に確率)を持ち、それ以外の範囲で確
率密度が0になる分布を一様分布\index{いちようぶんぷ@一様分布}と呼ぶ。もう少し形式的に表すと、確率密度関数\(f(x)\)が、

\begin{equation}
f(x)=\left\{
\begin{array}{ll}
0             & x < a\\
\frac{1}{b-a} & a \leq x \leq b\\
0             & b < x
\end{array}\right.
\label{eq_uniform102}
\end{equation}

\noindent
で表されるとき、\(f(x)\)を\(U(a,b)\)に従う一様分布と呼ぶ。

\begin{figure}
\begin{center}
\includegraphics[scale=0.6]{Figures/impulse1}
\end{center}
% \vspace{-2em}
\caption{一様分布}
\label{impulse1}
\begin{small}
\begin{quotation}
灰色の部分が各一様分布の確率密度関数と横軸の間の空間である。(a) \(U(0,1)\)、(b) \(U(0,0.5)\)、(c) \(U(0,0.1)\)
\end{quotation}
\end{small}
\end{figure}

さて、図\ref{impulse1}に\(U(0,p)\)となる一様分布を\(p=1,\ 0.5,\ 0.1\)について
示してある。仮に\(U(0,p)\)を表す確率密度関数を\(\upsilon_{p}(x)\)とすれば、\(p\)が
小さくなるに従って\(\upsilon_{p}(0)\)の値が大きくなっていくことが分かる。\(p\to 0\)の
極限では\(\upsilon_{p}\)はどのような関数になるだろうか。
そのような関数を{\bf ディラックの\(\delta\)関数}\index{でるたかんすう@デルタ関数}\footnote{インパルス関数とも呼ばれる。}と呼び、\(\delta(x)\)で表す。すなわち、
\(\delta(x)=\lim_{p\to 0}\upsilon_{p}(x)\)である。なお混乱を招く恐れがなければ、単にデルタ関数と呼ぶ。

まず図\ref{impulse1}を見て分かる
通り、灰色の部分の面積すなわち\(\upsilon_{p}(x) > 0\)となる\(x\)の範囲と\(\upsilon_{p}(0)\)の積
は\(p\cdot \frac{1}{p}\)である。\(p\to 0\)の極限をとると\(\lim_{p\to 0}p\cdot \frac{1}{p}=1\)
である。従って
\(\int_{-\infty}^{\infty}\delta(x)dx=\int_{-\infty}^{\infty}\lim_{p\to 0}\upsilon_{p}(t)dx=1\)
が成立する。また、式\ref{eq_uniform102}に\(a=0,\ b=p\)を代入し、\(p\to 0\)を計算することによって、
\(x < 0\)または\(x > 0\)のとき、\(\delta(x)=\lim_{p\to 0}\upsilon_{p}(t)=0\)が成立する。

% \begin{equation}
% \delta(x)=\lim_{p\to 0}\upsilon_{p}(t)=\left\{
% \begin{array}{ll}
% 0 & \mbox{for}\ x < 0\\
% 0 & \mbox{for}\ 0 < x\\
% \end{array}
% \right.
% \end{equation}

それでは\(\delta(0)\)はどのような値になるのだろうか？図\ref{impulse1}から見て取れるのは、
\(p\to 0\)のとき、灰色の部分の高さは限りなく高くなるということである。つまり
\(\delta(0)\)は\(+\infty\)に発散する。以上を以下の定理にまとめよう。

\vspace{1em}

\refstepcounter{theorem}
\noindent
{\bf 定理\thechapter-\thetheorem}: 以下の数式\ref{eq_delta_501}で定義されるディラックのデルタ関数は
式\ref{eq_delta_502}および式\ref{eq_delta_503}を満たす性質を持つ。

\begin{equation}
\delta(x)=\lim_{p\to 0}\upsilon_{p}(x)
\label{eq_delta_501}
\end{equation}

\begin{equation}
\delta(x)=\left\{
\begin{array}{ll}
0 & \mbox{for}\ x < 0\\
(+\infty\mbox{に発散}) & x = 0\\
0 & \mbox{for}\ 0 < x\\
\end{array}
\right.
\label{eq_delta_502}
\end{equation}

\begin{equation}
\int_{-\infty}^{\infty}\delta(x)dx=1
\label{eq_delta_503}
\end{equation}

これまで述べた通り、デルタ関数は一様分布が\(x=0\)のところに集中した極限とみなすことができるが、同時に
平均を0とする正規分布の分散が限りなく0に近づいた状態を表すと考えることもできる。すなわち、

\begin{equation}
\delta(x)=\lim_{\sigma\to 0}\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{x^{2}}{2\sigma^{2}}}
\end{equation}

\noindent
である。

\vspace{1em}

\refstepcounter{theorem}
\noindent
{\bf 定理\thechapter-\thetheorem}:~\label{delta_conv_theorem10} デルタ関数\(\delta(x)\)および連続な関数\(f(x)\)に関して以下の積分が成立する。

\begin{equation}
\int_{-\infty}^{\infty}f(x)\delta(x-\tau)dx=f(\tau)
\label{delta_conv102}
\end{equation}

\noindent
ちなみにこれを関数\(f(x)\)と\(\delta(x)\)の畳み込み積分\index{たたみこみせきぶん@畳み込み積分}という。

\noindent
{\bf 証明\thechapter-\thetheorem}: 積分\(\int_{-\infty}^{\infty}f(x)\upsilon_{p}(x-\tau)dx\)を考えると、
\(\upsilon_{p}(x-\tau)\)は\(0 \leq x-\tau \leq p\)すなわち
\(\tau \leq x \leq \tau+p\)の範囲でのみ\(\frac{1}{p}\)という値を持ち、それ以外では0である。
従って、

\begin{equation}
\int_{-\infty}^{\infty}f(x)\upsilon_{p}(x-\tau)dx=\int_{\tau}^{\tau+p}f(x)\upsilon_{p}(x-\tau)dx
\label{delta_conv103}
\end{equation}

\noindent
\(f(x)\)が連続であれば、\(p\to 0\)のとき、\(f(\tau)\simeq f(\tau+p)\)で一定値とみなせる
(コラム\ref{funccont1}参照)。
従って、

\begin{displaymath}
\int_{-\infty}^{\infty}f(x)\delta(x-\tau)dx
=\lim_{p\to 0}\int_{\tau}^{\tau+p}f(x)\upsilon_{p}(x-\tau)dx
=f(\tau)\lim_{p\to 0}\int_{\tau}^{\tau+p}\upsilon_{p}(x-\tau)dx
=f(\tau)
\end{displaymath}

\begin{flushright}
□
\end{flushright}

通常の関数であれば、\(x\)軸上のある1点\(x=a\)で0以外の値を取り(\(f(x) \neq 0\) for \(x = a\))、それ以外の点で
0の場合(\(f(x)=0\) for \(x \neq a\))、\(x=a\)を含む区間を定積分しても0になる。一方デルタ関数は\(x=0\)を含む点を
積分すると1が得られる。このような意味でデルタ関数は通常の関数とは一線を画するものであり、
{\bf 超関数}\index{ちょうかんすう@超関数}と呼ばれている。

デルタ関数はそれ単体で意味を持つ関数として用いられることより
式\ref{delta_conv102}のような他の関数との定積分ではじめて全体として意味を持つ関数として用いられることが多い。
特に式\ref{delta_conv102}は関数\(f(x)\)を入力として\(f(\tau)\)が出力される関数であると考えることができる。
このような関数を入力とするような関数を汎関数\index{はんかんすう@汎関数}と呼ぶ。
詳しくは節\ref{fourier}で説明しよう。

\begin{figure}
\refstepcounter{column}\label{col_hyperfunc}
\begin{itembox}[l]{コラム\thecolumn:超関数}
ここでは超関数の概要をごく簡単に説明しよう。まず以下の条件を満たす
関数を$\varphi(x)$を急現象関数と定義する。

\begin{enumerate}
\item $\varphi(x)$が何回でも微分可能
\item 各整数$l,m\geq 0$について常に$x^{l}\varphi^{(m)}(x) \leq \mbox{定数} <  \infty$
\end{enumerate}

\noindent
この定義に従うならば、例えば$e^{-x^{2}}$は急減少関数である。今$-\infty<x<\infty$において区分的に連続な関数
からなる列
\(u_{1}(x),u_{2}(x),\cdots,u_{n}(x),\cdots\)
がどんな急現象関数$\varphi(x)$についても有限な積分値

\begin{displaymath}
\int_{-\infty}^{\infty}u_{n}(x)\varphi(x)dx\ \ \ \ \ (n=1,2,\cdots)
\end{displaymath}

\noindent
を持ち、さらにどんな$\varphi(x)$に対しても

\begin{displaymath}
\lim_{n\rightarrow\infty}\int_{-\infty}^{\infty}u_{n}(x)\varphi(x)dx
\end{displaymath}

\noindent
が存在するとき、これを$u(\varphi)$で表し、$\varphi$から$u(\varphi)$への対応を表す汎関数$u$を超関数と呼ぶ。
すなわち、

\begin{displaymath}
u(\displaystyle \varphi)=\int_{-\infty}^{\infty}u(x)\varphi(x)dx=\lim_{n\rightarrow\infty}\int_{-\infty}^{\infty}u_{n}(x)\varphi(x)dx
\end{displaymath}

\noindent
と表現される。
\(U(0,p)\)を表す確率密度関数を\(\upsilon_{p}(x)\)とし、
$u_{n}(x)=\upsilon_{1/n}(x)$とすれば、$u(x)$はデルタ関数$\delta(x)$である。

% \varphi(x)はこの場合、テスト関数と呼ばれる。\delta(x)は
% あらゆる\varphi(x)に対する効果\varphi(0)の全体と考える。

\end{itembox}
\end{figure}
