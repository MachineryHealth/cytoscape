\section{概要}
\begin{description}

\item[研究目的] 
生物は細胞の中のｍＲＮＡという分子上の翻訳開始領域をどのようにして正
確に識別しているのか、その手ががりをつかむこと

\item[意義] 翻訳開始領域は合成されるタンパク質の性質を決める重要な箇所
\ \ \ →\ \ \ 生物学に対する多大な貢献

\item[研究方法] 
遺伝子データベースのコンピュータ解析

\item[研究成果] 
翻訳開始領域の識別に重要と思われる傾向をいくつか発見することができた

\item[Originality] 
研究手法の新しさ、新しい傾向の発見

\end{description}



\section{翻訳開始領域の上流に現れるＡＵＧの傾向の分析結果}

\subsection{分析１}
\begin{description}
\item[仮説]２つのＡＵＧが接近しているとリボソームは翻訳開始ＡＵＧの区別がつ
きにくくなるのではないか? 

\item[分析]翻訳開始領域の上流に現れるAUG配列の頻度の計算

\item[結果]翻訳開始領域直前でＡＵＧ配列の頻度が下がる(図 \ref{atg_freq1})

\item[考察] 翻訳開始領域直前のＡＵＧ配列は翻訳に大きな影響を与える
\end{description}

\subsection{分析２}
\begin{description}
\item[仮説]翻訳開始領域の認識能力と進化には関係がある
\item[分析]翻訳開始ＡＴＧとそこからみて上流に最初に現れるＡＴＧとの距
離の平均を計算
\item[結果]高等な生物ほど平均距離が長くなっている(図 \ref{atg_dod2_2})
\item[考察]下等な生物ほど翻訳開始領域の認識能力が高い
