\section{遺伝子とＤＮＡ}
\begin{itemize}
\item 遺伝子は生物個体の形質を決める因子
\item 遺伝子の情報は細胞の中のＤＮＡという高分子に書かれている
\item ＤＮＡの中で遺伝子として使われる領域はほんの一部
\item 生物はどのようにして遺伝子とそうでない部分を見分けているのか
\end{itemize}
