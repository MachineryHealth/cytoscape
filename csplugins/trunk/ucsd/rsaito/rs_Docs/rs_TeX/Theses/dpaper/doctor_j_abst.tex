本研究の目的は、(GenBankなど遺伝子データベース中にある)遺伝子の翻訳開
始領域に存在する配列を様々な角度からコンピュータで解析することによりそこ
に隠れた傾向を発見し、生物学的に考察することである。

翻訳開始とはmRNAと呼ばれる細胞内高分子がタンパク質に変換される最初のプロ
セスのことである。しかしそのプロセスに関しては不明な点が多く、mRNA上のど
のような配列とどのような因子が翻訳開始に関与しているのか、まだ完全には解
明されていない。そこで様々な生物のmRNAまたはそのもととなるDNAの配列を様々
な角度からコンピュータで分析し、配列中に隠れた傾向の抽出を行った。

その結果、以下のようなことが判明した。(a) 翻訳開始領域に出現するコンセン
サス配列はほとんどの場合、Shine-Dalgarnoの仮説やKozakの仮説に基づく配列
に一致する。しかし、{\it M.genitalium}ではそのような配列は見られなかった。
この種はUAAという配列パターンを翻訳開始シグナルの一部としている可能性が
ある。Shine-Dalgarno配列は翻訳の再開(reinitiation)の時にもおおむね必要であ
ることが判明した。(b) {\it E.coli}では、開始コドンがGTGになると
Shine-Dalgarno配列のコンセンサスが弱くなる。従ってこの種では開始コドンと
Shine-Dalgarno配列両方のコンセンサスを強くすることによって発現量の高い遺
伝子の翻訳開始の効率を上げていると思われる。(c)原核生物ではリボソーム結
合部位付近にコンセンサス配列を作りやすくするような塩基間の相互作用がある
ことが判明した。しかしこれは真核生物に関しては必ずしもあてはまらなかった。
また{\it H.sapiens}や{\it M.musclus}には開始コドン直前の2塩基を同じもの
にするような相互作用があることがわかった。(d)ATGという配列パターンが開始
コドン周辺ではリボソームの適切な翻訳開始反応を阻害する可能性があることが
コンピュータ解析により確かめられた。またATGの頻度の分布を様々な生物で比
較し、原核生物と真核生物では翻訳開始のプロセスが異なるという従来の学説を
サポートする結果を得ている。さらに古細菌と呼ばれる生物種は原核生物と真核
生物両方の翻訳開始のメカニズムを使っている可能性があることが判明した。

これらの結果は翻訳開始のプロセスを説明するものであり、将来的には実験等に
よって確かめられることが期待される。
