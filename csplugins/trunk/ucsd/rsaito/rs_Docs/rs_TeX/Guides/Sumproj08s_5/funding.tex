
\section{各種費用の補助}

\subsection{鶴岡までの交通費補助\index{こうつうひほじょ@交通費補助}}

\begin{itemize}
\item 「報告書」の提出と、SFCで（遠隔可）研究会時に「成果発表\index{せいかはっぴょう@成果発表}プレゼン」をしていただくことが、夏プロ旅費補助の条件です。
（報告書についての期限など詳細は本ガイドの\ref{report}節を参照してください）

\item 今回初めて夏プロに参加する人はゼロ\footnote{今回夏プロが初めてでも、以前春プロに参加していればOK。}。

\item 交通費の補助は往復１回分のみとなります(例：1陣と3陣参加の場合でも一往復分のみ)。

\item JRのチケット、航空券はこちらで手配します(※立て替え不可)。
   \begin{itemize}
   \item 事前に日程／列車名･便名／年齢をご連絡下さい。（締切日：2008年7月11日（金）までにmorimami@sfc.keio.ac.jpまで）\\
※注意：締切日以降の変更キャンセルは不可
   \item JR利用の場合は学割を適用。「学生・生徒旅客運賃割引証」を提出して下さい。
   \item 飛行機の場合は半券が必要となりますので失くさないよう注意して下さい。
   \item 交通費補助の区間はJRまたは航空運賃のみとなります(最寄り駅からの私鉄・バス運賃はなし)。
   \item 旅費申請書については追ってご連絡致します。\\
※事前に、交通費補助を受ける旨の連絡がない場合、また個人の立て替えでチケットを
購入した場合は自費となりますので予めご了承下さい。 
   \end{itemize}
\end{itemize}


\subsection{宿泊費について\index{しゅくはくひ@宿泊費}}

\begin{itemize}
\item 研修棟の宿泊費は参加者全員に補助されます。
\end{itemize}

\subsection*{\colorbox{red}{注意!}}

以下の場合は旅費補助の対象外となりますのでご留意ください。

\begin{itemize}
\item 正当な理由なく、チェックイン・チェックアウト日を変更した場合。
\item 正当な理由なく、プロジェクト日程を変更したり、プロジェクト期間中に研究場所にいなかった場合。
\item 正当な理由なく、報告書を締め切りまでに提出しなかったり、内容が著しく貧弱な場合。
\item 正当な理由なく、予定された成果発表プレゼンを怠った場合。
\item その他必要な書類を締め切りまでに提出しなかった場合。
\end{itemize}
