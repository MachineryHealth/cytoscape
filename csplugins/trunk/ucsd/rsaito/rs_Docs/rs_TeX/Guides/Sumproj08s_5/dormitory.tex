\subsection{利用方法}

\begin{description}
\item[ふとん] シーツ、カバーは、1週間に1回交換できます(1週間未満の滞在の場合、交換不可)。
交換用のシーツ、カバーは、2階踊り場に置いてありますので、各自で取り替えてください。
使用済のシーツ、カバーは、2階踊り場のリネン箱に入れてください。
\item[ゴミ] 各部屋のゴミは、共用スペースのゴミ箱に捨ててください。清掃業者が回収します｡
シンクのゴミはすぐに片付けてください。
\item[火気]火の元には十分気をつけてください。
特に、ガスコンロの扱い、ポット・炊飯器の空焚きに注意してください｡
\item[タバコ]研修棟内は禁煙です。玄関脇に灰皿があります。
\item[鍵・カードキー] 各人が責任を持って管理してください。紛失の場合は、実費を請求します。
\item[退室時の注意] 各部屋・共有スペースを、入室時の状態に戻して、退出してください。
各部屋・共有スペースの冷蔵庫を空にしてください(冷蔵庫のスイッチは切らないでください。
ふとんのシーツ、カバーをはずしてリネン箱に入れてください\footnote{新しいカバーをつける必要はありません。}。
\end{description}

\noindent
省エネルギーにご協力ください。
\begin{itemize}
\item 部屋を出るときは、照明・冷暖房をオフに
\item 冷暖房は過度の温度設定を避けて（冷房時28℃以上、暖房時20℃以下）
\end{itemize}
