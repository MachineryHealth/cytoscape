
\section{プロジェクト成果報告要領}
\label{report}

\subsection{報告書の内容\index{ほうこくしょ@報告書}}

報告書の言語は英語または日本語とします。{\bf 報告書の内容が充分でなかったり、
締め切りなどを守らないプロジェクトには、交通費や研修棟費用を支給いたしま
せんのでご注意ください。}構成は以下のとおりです。

\begin{description}
\item[はじめに・背景・役割分担など] 2ページ。単位を申請する学生が手分けして書いてください。
\item[各メンバーの要旨一覧] 各メンバーの要旨（5、6行）だけをずらっと並べる。特に良い成果が出たと思
うものには要旨の最初に[プレミアプレゼン希望]と明記する。
\item[各メンバーのページ] 要旨（5、6行）を含む。単位を申請する学生は4〜6ページ、それ以外の学生
は2〜3ページとする。単位を申請しない学部生はその旨報告書の中で宣言して
ください。ただし投稿論文がある場合はそれをそのまま提出して報告書にかえる
ことができます。
\end{description}

\subsection{報告書の提出}

参加するプロジェクト1つごとに、提出義務の有無が決まります。

\begin{enumerate}
\item \label{rep1} 単位申請の対象となるプロジェクト
\item \label{rep2} 旅費補助申請の対象となるプロジェクト
\end{enumerate}

上記\ref{rep1}, \ref{rep2}に該当する全件について、1名で複数該当する場合
にも、報告書を提出してください。このほかに、今回はじめて先端生命科学研究
会の特別研究プロジェクトに参加する学生は、少なくとも1報の報告書を提出し
てください。

\vspace{1em}

※以上は「義務」の話です。もちろん、参加するプロジェクトすべてについて報告書を提出できますし、それを期待しています。

\vspace{1em}

研究会webサイト（apple.bioinfo.ttck.keio.ac.jp）よりPDF形式で提出してく
ださい（タームペーパーと同様）。締め切りは9月26日(金) 15:00です。提出用
のURLなどは追ってアナウンスします。

\subsection{成果発表プレゼン}

夏プロの成果発表はSFC（遠隔可）で秋学期授業時間に開催予定です。単位また
は旅費を申請する学生は全員、成果発表プレゼンをしていただきます。時間は1
人5分です。これ以外に冒頭で、グループ全体の概要を最大3分まで話していた
だいて結構です（プレゼンターは誰でもOK）。どうしてももっと時間がほしい人
はその旨理由とともに（グループリーダーから了承をとった上で）rsaitoまでお
知らせください。
