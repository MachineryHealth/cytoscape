\subsection{Systems Immunology}
\begin{description}
\item[種別(D/W)]
\item[プロジェクトリーダ] M. Tsuchiya \& K. Selvarajoo
\item[連絡先] TTCK
\item[実施期間]	8/1-9/16
\item[実施場所] TTCK
\item[参加条件]	1陣〜3陣(最低２陣は必修)
\item[要旨] 本研究グループでは、病原体の侵入から、病原体認識の自然免疫から、病原体の
抗原情報を記憶化、排除する獲得免疫への活性化を指揮するT細胞の活性化・分
化のダイナミックなメカニズムを解明するため、シグナル伝達経路と遺伝子発現
制御機構を統合したダイナミックなシステム生物学的手法を開発しています。、
自然免疫担当細胞（マクロファージ、樹状細胞）の実験データの解析に、これら
の手法を応用し、免疫システムにおける情報伝達の制御や調整のしくみの解明、
医療への応用を目指しています。 自然免疫活性化のダイナミックなメカニズム
の解明は、具体的な免疫制御の可能性、感染症に対する効果的なワクチン開発、
免疫異常による疾患の治療法、キラー細胞によるガン免疫療法の開発などの具体
的な医療応用への実現の可能性が期待されます。これらの研究は、阪大微生物研、
慶應医学部、東大医科研、Caltech（USA), Italian NIH との共同研究です。更
に、免疫の誕生に重要な役割を果たす、自己・非自己を認識する受容体・MHC
（主要組織適合複合体）の分子進化の研究も行っています。

      \item TLRシグナル伝達経路のネットワークモデルの構築、ネットワークの動的活性化のシュミレーション　（アドバイザー：　Kumar Selvarajoo）
      \item 転写因子並びRNAサイレンシングによる遺伝子発現のダイナミック制御モデルの構築とシュミレーション　（アドバイザー：Masa Tsuchiya）

\subsection{ヌクレオソームとメチル化の関係の大規模解析}
\begin{description}
\item[種別(D/W)] D
\item[プロジェクトリーダ] 中津大貴
\item[連絡先] nakatsu@sfc.keio.ac.jp
\item[実施期間] 1陣
\item[実施場所]	鶴岡
\item[参加条件]	ヌクレオソームポジショニングとメチル化の基礎知識を持ち、本解析に必要と思
われるプログラミング能力を有すること
\item[要旨]
DNAのメチル化とヌクレオソームポジショニングは深い関係にあると、先行研究
により示唆されている。しかし、これまで双方の関係を大規模に解析したという、
報告は行なわれていない。その原因として、DNAのメチル化とヌクレオソームポ
ジショニングともこれまで、大規模なデータが得られていなかったことが挙げら
れる。しかし、2004年にメチル化データ、2008年5月にはヌクレオソームポジシ
ョニングの大規模データが公開された。本プロジェクトでは、この2つの大規模
データを解析することで、ヌクレオソームとメチル化の関係についての新たな知
見を得ることを目指す。
\item[実施計画] \verb+ +
   \begin{description}
   \item[第1日目] テーマ全体の流れを決定するミーティング
   \item[第2〜3日目]  2つのデータの統合と基礎解析
   \item[第3〜10日目] 統合データの解析と解析結果に基づく考察
   \item[第11日目] 報告書の作成
   \end{description}
\end{description}

HII/2と酷似しているので、必要に応じてこれらとの比較解析も行う。特定アミ
ノ酸のRNase HあるいはJRNase活性への寄与を明らかにすることを目的とする。
\item[実施計画] \verb+ +
   \begin{description}
   \item[第1日目] 実験方針の説明およびそのディスカッション、試薬の準備
