This screenshot shows the resultsof running MCODE on a set of budding yeast protein-protein interaction data. MCODE found a number of clusters, shown in each row of the result table. Clicking on the individual rows will either highlight that cluster in the original network or create a new network containing only that cluster, depending on the option selected at the bottom of the results window.
