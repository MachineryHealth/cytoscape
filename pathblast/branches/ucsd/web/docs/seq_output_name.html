Specifying proteins in the pathway

You can input either a Protein ID or Protein Sequence for each of your proteins.
  • If you choose to use Protein ID, you don't need to fill in the Protein Sequence field. Your Protein ID will be used to automatically retrieve the protein sequence and label your protein in the graphical overview.
  • If you choose to use Protein Sequence, it will be used for the query and a name for the sequence will be assigned (e.g. "A").
  • If you enter both, the sequence will be used and the Protein ID will be used as teh sequence name.
